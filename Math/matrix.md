## 1. 定理

**1. m $\times$ n 矩阵: ** m行n列的实数构成的矩阵阵列

- 元素以$M_{ij}$表示
- 行向量(x, y, z): $1 * n$的矩阵
- 列向量[x, y, z]: $m * 1$的矩阵

**2. 矩阵的一般运算**

- 矩阵相等: **当且仅当两个矩阵的对应元素相等**
- $m \times n$ 矩阵之间的加法: 两者对应元素相加
- 标量乘法: **标量依次和每个元素相乘**
- $m \times n$ 矩阵之间的减法: $A - B = A + (-1 \cdot B) = A + (-B)$

- 加法结合律: $(A + B) + C = A + (B + C)$
- 标量乘法对矩阵加法的分配律: $r(A + B) = rA + rB$
- 矩阵乘法对标量加法的分配律: $(r + s)A = rA + sA$
